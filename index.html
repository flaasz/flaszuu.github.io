Guardian - stay safe!

Guardian is a JavaScript based bot for Discord. Playing with bots on Discord is fun, but coding them is much better. This bot is meant to be some kind of server guard, but now i think of adding some fun stuff, such as exp system or some sort of gamling.

You can understand how bots work, and improve your Node.js skills pretty easily. I totally recommend trying it out youself!

Note, that this bot works in Polish, not English.
